# ニューラルネットの直感 ── y = f(Wx + b) が全ての出発点

> シリーズ「LLMの中身を数式で追う」第2回

---

前回、行列はベクトルを変換する操作だと学んだ。  
今回は、その行列を使って「学習する機械」をどう作るかを見る。

---

## 最もシンプルなニューラルネット

ニューラルネットを一言で表すとこうだ。

```
y = f(Wx + b)
```

記号が並んでいるが、一つずつ見れば難しくない。

| 記号 | 意味 |
|---|---|
| `x` | 入力ベクトル（与えるデータ） |
| `W` | 重み行列（学習で調整される数値） |
| `b` | バイアスベクトル（調整用のオフセット） |
| `f` | 活性化関数（後述） |
| `y` | 出力ベクトル（予測結果） |

`Wx + b` は「行列 W で x を変換して、b を足す」だけだ。  
前回の行列積に、数値を足す操作を加えた。

---

## 「重み」とは何か

`W` の中の数値を「重み（weight）」と呼ぶ。

具体例で考える。「気温」「湿度」から「傘が必要か（0〜1）」を予測するとする。

```
x = [気温, 湿度] = [30, 80]  ← 暑くて湿度高い

W = [[-0.05, 0.02]]  ← これが重み

Wx = -0.05×30 + 0.02×80 = -1.5 + 1.6 = 0.1
```

この `W` の中の数値 `-0.05` と `0.02` が重みだ。  
「気温が高いほど傘は不要（マイナスの重み）」  
「湿度が高いほど傘が必要（プラスの重み）」  
という関係を数値で表している。

**学習とは、この重みの数値を少しずつ調整すること。**  
最初は乱数で初期化して、正解データと比較しながら「より正確な予測ができる重み」に近づけていく。

---

## 「活性化関数」とは何か

`y = Wx + b` だけだと、どれだけ行列を重ねても「線形変換の連鎖」にしかならない。  
線形変換をいくら重ねても、結局1つの線形変換と同じ結果になる。

これでは複雑なパターンを学習できない。

そこで `f`（活性化関数）を間に挟む。  
最もよく使われるのが **ReLU（Rectified Linear Unit）**：

```
ReLU(x) = max(0, x)

 入力が正 → そのまま通す
 入力が負 → 0 にする
```

図で見ると：

```
          /
         /
________/
       0
```

シンプルだが、これを挟むことで「非線形」な変換が可能になる。  
非線形 = 「曲がった境界線」で分類できる = 複雑なパターンを学習できる。

---

## 層を重ねる

`y = f(Wx + b)` を1層とすると、これを複数重ねたものが「深い（Deep）ニューラルネット」だ。

```
x → [f(W₁x + b₁)] → [f(W₂h₁ + b₂)] → [f(W₃h₂ + b₃)] → y
         第1層               第2層               第3層
```

各層が「前の層の出力を変換する」を繰り返す。  
層が深くなるほど、より抽象的な特徴を学習できる。

- 第1層: 単純なパターン（線、角など）
- 第2層: パターンの組み合わせ
- 第3層: より高次の概念

LLM では、この「層を重ねる」構造を Transformer と呼ばれる特殊な設計で実現している。

---

## 「学習」の正体

重みをどうやって調整するのか。流れはこうだ。

```
1. 入力 x を与えて予測 y を出す
2. 正解 y_true と比較して「どれくらい外れたか（損失）」を計算
3. 損失を小さくする方向に W を少しだけ調整（勾配降下法）
4. 1〜3 を何百万回も繰り返す
```

「少しだけ調整」の量を学習率（learning rate）と呼ぶ。  
これが大きすぎると正解を通り過ぎ、小さすぎると学習が遅い。

LLM の「学習」は、数兆トークンのテキストに対してこれを繰り返した結果だ。  
GPT-3 の W の数値（パラメータ）は約 1,750 億個。これを全て調整していく。

---

## まとめ

| 概念 | 正体 |
|---|---|
| ニューラルネット | `y = f(Wx + b)` の繰り返し |
| 重み W | 学習で調整される行列の数値 |
| 活性化関数 f | 非線形性を導入する関数（ReLU等） |
| 学習 | 正解との差を縮める方向に W を調整し続けること |
| 深いネット | 上記を何十層も重ねた構造 |

次回はいよいよ LLM の全体像を一枚絵で追う。  
これまで学んだ「ベクトル」「行列積」「層を重ねる」がどう組み合わさっているかを見る。
